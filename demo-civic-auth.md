# 🎬 PeerLink Civic Auth Demo Script

## 🎯 Demo Overview
Showcase how PeerLink uses Civic Auth for instant, trusted connections without oversharing personal information.

## 🚀 Demo Flow

### 1. **Landing Page** (0:00 - 0:30)
- **Show:** Beautiful landing page with PeerLink branding
- **Highlight:** "Instant trusted connections — without oversharing"
- **Click:** "Login with Civic Auth" button
- **Explain:** "This starts the Civic Auth flow for verified identity"

### 2. **Civic Auth Flow** (0:30 - 1:00)
- **Show:** Civic Auth redirect/authentication
- **Explain:** "Civic Auth verifies the user's identity without revealing sensitive data"
- **Result:** User gets verified and redirected back to PeerLink

### 3. **Dashboard** (1:00 - 1:30)
- **Show:** User dashboard with profile info
- **Highlight:** "Civic Verified" badge
- **Explain:** "User is now authenticated with verified identity"

### 4. **Profile Management** (1:30 - 2:00)
- **Navigate:** Click "Profile" tab
- **Show:** Profile editing interface
- **Highlight:** Privacy toggles for each field
- **Explain:** "Users control exactly what information to share"

### 5. **QR Code Generation** (2:00 - 2:30)
- **Navigate:** Click "My QR" tab
- **Show:** Generated QR code
- **Explain:** "This QR code contains a link to their public profile"
- **Demo:** Scan QR code or show share link

### 6. **Privacy Controls** (2:30 - 3:00)
- **Show:** Toggle visibility settings
- **Explain:** "Email is private, LinkedIn is public, wallet is private"
- **Demo:** Show how public profile changes based on settings

### 7. **Event Management** (3:00 - 3:30)
- **Navigate:** Click "Events" tab
- **Show:** Event creation and management
- **Explain:** "Organizers can create events with QR codes for attendees"

### 8. **Messaging** (3:30 - 4:00)
- **Navigate:** Click "Messages" tab
- **Show:** Private messaging interface
- **Explain:** "Users can message each other without exchanging contact info"
